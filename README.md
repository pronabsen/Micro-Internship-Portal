# Micro-Internship Portal

A comprehensive web-based platform connecting students with short-term internship opportunities and enabling employers to recruit talent for specific tasks. This project is a frontend-only implementation using HTML, CSS, and JavaScript with dummy data, simulating a full-stack experience.

## ğŸŒŸ Features

### For Students
- **Registration & Authentication**: Create an account and log in to access the platform
- **Profile Management**: Build a comprehensive profile with skills, education, and resume
- **Browse Internships**: Search and filter internships by category, duration, and stipend
- **Apply for Opportunities**: Submit applications with cover letters and availability details
- **Track Applications**: Monitor the status of your applications in real-time
- **Earn Points & Badges**: Gamified experience with points, ranks, and achievements
- **Receive Reviews**: Get rated by employers after completing internships

### For Employers
- **Company Registration**: Register your company and get verified by administrators
- **Post Internships**: Create detailed internship postings with requirements and stipend
- **Manage Applications**: Review applications from students and accept/reject candidates
- **Rate Interns**: Provide feedback and ratings to interns after project completion
- **Company Profile**: Maintain and update company information

### For Administrators
- **Dashboard Overview**: Monitor platform statistics and user activity
- **Employer Verification**: Approve or reject employer registration requests
- **User Management**: View and manage all user accounts
- **Content Moderation**: Review and remove inappropriate internships
- **Generate Reports**: Access analytics and insights on platform usage

## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, Safari)
- No additional software or installations required

### Installation
1. Clone the repository or download the source code
2. Open the `index.html` file in your web browser
3. The application will load with dummy data pre-populated

### Quick Start
1. **As a Student**:
   - Click "Get Started" or "Register" to create a student account
   - Complete your profile with education and skills
   - Browse available internships and apply to ones that match your interests

2. **As an Employer**:
   - Register as an employer during the registration process
   - Wait for admin verification (for demo purposes, you can login directly)
   - Post internships and review applications

3. **As an Admin**:
   - Login with admin credentials (use any employer account for demo)
   - Access the admin dashboard to manage the platform
